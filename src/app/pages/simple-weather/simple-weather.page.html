About This App

This is a demonstration of a NASA Weather Explorer app built with Ionic Angular.

The app is designed to:

  • Allow users to select locations via map or text input
  • Choose weather variables to analyze (temperature, precipitation, wind, etc.)
  • Select date ranges for historical analysis
  • Display interactive charts and statistical analysis
  • Export data in various formats

Data Sources: NASA MERRA-2, GPM, MODIS datasets
